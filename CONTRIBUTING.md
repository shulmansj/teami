# Contributing

Thanks for taking a look at Agentic Factory.

This repository is public so people can evaluate the product direction,
implementation quality, and setup experience. Best-effort feedback is welcome
through GitHub Issues.

External pull requests are not supported yet. The project is still using a
curated source-visible launch process, so maintainers may read feedback and
apply changes privately instead of reviewing or merging community PRs.

## Good Feedback

- Setup bugs with exact commands, environment details, expected behavior, and
  what happened instead.
- Product feedback about what is clear, confusing, missing, or not credible.
- Product ideas framed around the user, the workflow, and the outcome they need.

## Security Reports

Do not report vulnerabilities in public issues. Use GitHub private vulnerability
reporting as described in `SECURITY.md`.
