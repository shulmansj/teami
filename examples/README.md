# Examples

Provenance: hand-curated from current output.

Examples show how Agentic Factory artifacts should read without requiring a
visitor to connect a real Linear workspace.

## Portfolio Demo

- [Portfolio demo](portfolio-demo/README.md): a synthetic B2B product-ops
  scenario showing a Linear roadmap item decomposed into agent-ready execution
  issues, a project update, and an eval-backed improvement proposal.

The portfolio demo uses fictional companies, people, account names, and run
identifiers. It is not roadmap truth for any adopter, and it does not claim
account-connected setup proof.
